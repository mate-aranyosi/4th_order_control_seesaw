# 4th_order_control_seesaw
A seesaw beam for demonstrating and experimenting with 2nd and 4th order control via one 2nd order system nested within another. The position of a mass along the beam's length controls its tilt (inner loop 2nd order system), which in turn controls the position of a ball rolling on top of the beam (outer loop 2nd order system).

This repository includes all documentation required to build and operate the 4th-Order Control Seesaw Beam (4OCSB, colloquially known as the Yeehaw Seesaw). Key items include:

1. CAD files & BOM 
2. Electrical schematics
3. Arduino and MATLAB code
4. Assembly instructions
5. Operating instructions
6. Technical report
7. Sample datasets

I hope this project will serve as an interactive platform for better understanding linear control systems and how certain modeling assumptions about them fare in real-world physical systems.
